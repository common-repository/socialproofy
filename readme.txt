== Installation ==

Social Proofy plugin installation is as as easy as it gets, it works with both WordPress and WooCommerce on your websites.
Social Proofy automatically connects to your WordPress website and WooCommerce store and pulls orders and signups as they happen.

Instructions:

1. Install the plugin through the WordPress plugins screen OR unzip the plugin archive to '/wp-content/plugins'
2. In the WordPress 'Plugins' page, activate the plugin.
3. Select "Social Proofy" menu item from your admin sidebar.
4. Copy over your API & Site key from Social Proofy dashboard and hit "Save".
5. In the Social Proofy dashboard hit "New Notification" and follow the wizard to create a "Latest Conversion" WooCommerce notification.
6. Done. Sit back and watch how social proof boosts your sales!

Notice:

* Social Proofy inject the Social Proofy javascript code to your website to display notifications and collect visitor data.

* Social Proofy is a cloud based service and the plugin communicates with the Social Proofy servers, including remote webhook calls, data collected includes product details and customer details.
